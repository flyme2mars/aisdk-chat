# Technology Stack: AI SDK Chat Demonstration

## Core Technologies
- **Language:** TypeScript
- **Frontend Framework:** Next.js 16 (App Router)
- **Library:** React 19

## UI & Styling
- **CSS:** Tailwind CSS v4
- **Components:** Radix UI primitives (shadcn/ui convention)
- **Icons:** Lucide React

## AI Integration
- **Framework:** Vercel AI SDK v6
- **Provider:** OpenAI Provider (configured for OpenRouter)
- **Streaming:** Real-time text streaming protocol
